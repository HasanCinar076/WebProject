# Webpage Design

This project showcases a responsive web design using HTML and CSS. It features a modern layout with an orange color theme and is designed to provide an intuitive user interface.

## Features

- **Responsive Navbar**: Includes links to Home, About, Service, Career, and Contact pages.
- **Search Functionality**: Integrated search bar within the navbar.
- **Dynamic Content Section**: Main content area with promotional text and a call-to-action button.
- **Login Form**: Includes a login form with email and password fields, a login button, and a link for new users to sign up.
- **Social Login Options**: Placeholder for social media login integration.

## Technology Stack

- **HTML**: For structuring the web page.
- **CSS**: For styling all components, with a focus on responsiveness and modern design aesthetics.
